# 智能抢单决策助手

一个基于Python的智能抢单决策系统，帮助用户自动化处理抢单逻辑。

## 技术栈

- **GUI框架**: PyQt6
- **HTTP客户端**: aiohttp  
- **音频播放**: playsound
- **开发环境**: Python 3.x + Windows

## 项目结构

```
抢单提醒/
├── venv/              # Python虚拟环境
├── rules.json         # 规则配置文件
├── main.py           # 主程序文件
├── upload.bat        # Git自动上传脚本
└── README.md         # 项目说明文档
```

## 快速开始

1. 激活虚拟环境：
   ```powershell
   venv\Scripts\Activate.ps1
   ```

2. 运行程序：
   ```powershell
   python main.py
   ```
